This System does not support Printing of the screen and the printing has to be done through Print Screen option.

In common browsers' system view cannot be printed entirely, in such cases, please use the Print Screen function available with the operating system.

Using Print Screen under various OS

1. Micorosoft Windows - Please follow the instructions below
Print Screen key only - To print the entire screen
Print screen and Alt key - Keep alt key down and press "print screen" to print an active window.

2. Unix or Linux - Please follow the instructions below
Print Screen key only - To print the entire screen
Print screen and Alt key - Keep alt key down and press "print screen" to print an active window.
Caution - On certain keyboards Alt key is presented as fn key and Print Screen key is presented as Print scrn etc.

3. MAC
Command + shift + 3 - To print the entire screen
Command + shift + 4 - To print an active window
Command + shift + 4 + caps lock - Picture is being saved.
Caution - For keys '3' and '4' ten keys cannot be used.
In case if above mentioned buttons are not available please use free image capture software's. These free wares are available on the web.
